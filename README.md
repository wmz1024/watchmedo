# Watch Me Do

在线视奸👁我的电脑 服务端

Rust编写 探针

## 功能特性

- 🖥️ **系统监控** - 实时监控CPU、内存、硬盘和网络使用情况
- 📊 **进程管理** - 查看正在运行的应用程序及其资源占用
- 🌐 **远程推送** - 将系统信息推送到远程服务器
- ⚙️ **设置管理** - 自定义应用程序配置和推送间隔
- ℹ️ **关于页面** - 查看应用信息、技术栈和联系方式

## 技术栈

- **Tauri** - 跨平台桌面应用框架
- **React** - 用户界面库
- **TypeScript** - 类型安全的JavaScript
- **Tailwind CSS** - 实用优先的CSS框架
- **Rust** - 系统后端语言

## 开发

```bash
# 安装依赖
npm install

# 开发模式
npm run tauri dev

# 构建应用
npm run tauri build
```
